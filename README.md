# Projeto de Banco de Dados — Clínica Médica (MySQL)

Script SQL para criação e teste de um banco de dados de **Clínica Médica**.

## Arquivo
- `script.sql`: cria as tabelas, relacionamentos, insere dados de exemplo e roda consultas (`SELECT`).

## Tabelas
- `especialidade`, `medico`, `paciente`
- `medico_especialidade` (relação N:N)
- `horario_disponivel`
- `agendamento`

## Como executar
- Rode o `script.sql` em um MySQL/MariaDB (Workbench, DBeaver ou terminal).
- Teste online (SQL Fiddle): https://sqlfiddle.com/mysql/online-compiler?id=42619709-0c99-4c0a-87ea-8ea0b121a26c
